# Agent Guide

You are working on a Ruby on Rails project.

Defaults:

- One Rails monolith
- Root domain = landing pages
- app subdomain = authenticated product
- Start new projects with the latest stable Ruby and Rails versions
- Keep solutions simple
- Use Rails conventions first
- Use generators when possible
- Ask before inventing product behavior
- If priority is unclear, stop and ask
- Responsive UI (desktop strong, mobile compatible)
- Add minimal tests
- Run RuboCop and tests before commit
